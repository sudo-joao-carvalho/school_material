This course comes with a couple of extra resources:

  • Code examples & snapshots: Attached to the lectures they belong to - and always to the last lecture of every module

  • Slides: Attached to the last lecture of every module

  • Summary Sheets (PDF documents): Attached to the last lecture of every module

In addition, you find all summary sheets combined into one single PDF document attached to this lecture.

"Attached" means that a link to the code / attachments is provided. You can then access the resources in the browser or download them from Github.
